Como jogar?

O Bomberman exige a participação de dois jogadores:

JOGADOR 1

JOGADOR 2

Tecla "A" : Esquerda

Tecla "W" : Para cima

Tecla "S" : Para baixo

Tecla "D" : Direita

Tecla "Espaço" : Disparar a bomba

Tecla "←" : Esqueda

Tecla "↑" : Para cima

Tecla "↓" : Para baixo

Tecla "→" : Direita

Tecla numérica "0" : Disparar a bomba

Objetivo

Vencerá o jogo quem explodir seu adversário mais vezes, entretanto, existem obtáculos (blocos), dos quais devem ser explodidos para que um jogador tenha acesso livre ao outro.

É necessário utilizar a bomba, tanto para matar o adversário, quanto para explodir os blocos.
